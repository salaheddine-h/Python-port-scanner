# Python Port Scanner

A simple Python-based port scanner that scans ports 1–1024 on a target host.

## Features
- Scans common ports
- Domain to IP resolution
- Displays open ports
- Fast socket-based scanning

## Technologies Used
- Python
- Socket Library

## Usage

python3 port_scanner.py

Enter target domain or IP when prompted.
